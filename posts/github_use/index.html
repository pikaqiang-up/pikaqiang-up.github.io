git基本设置#

  • 第一次使用时的配置
1#设置全局用户
2git config --global user.name "你的名字"
3git config --global user.email "你的邮箱"
4#初始化
5git init
  • 添加远端
1#添加
2git remote add origin <远程地址>
3#修改地址
4git remote set-url origin <地址>
5#查看地址
6git remote -v

基本操作#

  • 克隆
1#克隆
2git clone <仓库地址>
3#设置克隆深度
4git clone --depth=<层数> <仓库地址>
  • 添加到暂存区
1git add <文件>
2#技巧
3git add .       #添加当前文件夹
4git add *       #添加当前文件夹内的所有文件
  • 提交
1git commit                   #会打开editor编辑提交，信息
2git commit -m "提交信息"      #直接在命令行内写提交信息
3git commit -a                #自动add当前文件夹内所有文件
4git commit -am               #混合使用
  • 分支操作
1#查看本地所有分支
2git branch
3#查看远端所有分支
4git branch -r
5#添加分支
6git branch <分支名>
7#删除分支
8git branch -d <分支名>
